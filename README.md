## 데이터 분석을 위한 (종속/독립)변수를 추출하고 전처리(결측치, 이상치 정제)와 다양한 가설을 검정하고 시각화를 진행

# 필수
- 데이터 수집
- DF 타입 확인
- 기초통계량 확인(describe)
- 결측치/이상치 확인 및 정제
    - 레그플롯(이상치확인)
    - 박스플롯(이상치확인)
    - 히스토그램(이상치확인 및 도수 분포표 확인)
- 신뢰구간 확인
- 표준화(scaling)
- 주성분분석(PCA)
- 데이터 분석(상관분석, 회귀분석 등)
    - 데이터의 관계를 파악하고 해석
